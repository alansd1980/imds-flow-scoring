About This Tool

This tool implements the ELN iMDS flow cytometry scoring framework for bone marrow immunophenotyping in the diagnostic work-up of suspected MDS.

Five modules are assessed: the Ogata diagnostic score, erythroid aberrancy score, myeloid progenitor aberrancy assessment, granulocyte dysplasia features, and monocyte aberrancy assessment.

The integrated conclusion uses ABC scoring: Category A = no MDS-related immunophenotypic features; Category B = limited immunophenotypic changes associated with MDS; Category C = immunophenotypic features consistent with MDS.

This tool is for use by trained haematology laboratory scientists and clinicians. It does not replace expert morphological review, cytogenetic analysis or clinical judgment. No patient data is stored or transmitted.

Panel Requirements
  • CD45, CD34, CD117, CD13, CD33, CD11b, CD15, CD16, CD10
  • CD71, CD36, CD235a (erythroid lineage)
  • CD14, HLA-DR (monocyte lineage)
  • CD7, CD56, CD19, CD5, CD2, CD123 (aberrant antigens)
  • CD38, CD19, CD10, TdT (B-cell progenitors)
