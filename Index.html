About This Tool

This tool implements the ELN iMDS (integrated Myelodysplastic Syndrome) flow cytometry scoring framework for bone marrow immunophenotyping in the diagnostic work-up of suspected MDS.

The scoring system incorporates five modules: the Ogata diagnostic score, erythroid aberrancy score, myeloid progenitor aberrancy assessment, granulocyte dysplasia features, and monocyte aberrancy assessment. The integrated conclusion is reached when ≥2 modules are classified as abnormal.

Cutoff values are adjustable in the Configure Cutoffs tab to allow local validation and adaptation as ELN guidance evolves.

This tool is intended for use by trained haematology laboratory scientists and clinicians. It does not replace expert morphological review, cytogenetic analysis or clinical judgment.

Panel Requirements

For full scoring, the following markers are required:

  • CD45, CD34, CD117, CD13, CD33, CD11b, CD15, CD16, CD10
  • CD71, CD36, CD235a (erythroid lineage)
  • CD14, HLA-DR (monocyte lineage)
  • CD7, CD56, CD19, CD5, CD2, CD123 (aberrant antigens)
  • CD38, CD19, CD10, TdT (B-cell progenitors)
